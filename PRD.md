# PRD: SysMonWidget (Windows 항상-위 시스템 성능 모니터)

## 1. 개요 및 목표

**제품명(가칭):** SysMonWidget

**목적:** Windows 데스크톱에서 CPU, 메모리, GPU, 디스크/네트워크 I/O를 실시간으로 확인할 수 있는,
항상 다른 창 위에 떠 있는(Always-on-top) 초경량 모니터링 위젯.

**타겟 사용자:** 본인(1인 개인 사용). 배포/설치 마법사 등은 불필요하며 포터블 실행 파일 형태로 충분하다.

**핵심 가치:** 작업관리자를 열지 않고도 화면 한 켠에서 시스템 부하를 즉시 인지할 수 있어야 한다.
위젯 자체가 시스템 자원을 많이 소모해서는 안 된다.

## 2. 핵심 기능

### A. 미니 위젯 (기본 모드)
- 화면 위 항상-최상단(Topmost), 드래그로 위치 이동 가능
- CPU / RAM / GPU / 디스크·네트워크 I/O를 숫자로만 간결하게 표시
- 각 지표는 임계값 초과 시 색상 변경 (정상=기본색, 경고=노랑, 위험=빨강)
- 반투명 배경, 투명도 사용자 조절 가능

### B. 팝업(확장 뷰)
- 위젯 클릭/토글 시 팝업으로 전환
- 위젯과 동일한 지표를 표시하되, 좀 더 여유 있는 레이아웃 + 동일한 색상 경고 로직
- 팝업 내에서 임계값(경고/위험 기준) 수치를 사용자가 직접 조정 가능

### C. 경고 임계값 (기본값, 사용자 조정 가능)

| 지표 | 경고(노랑) | 위험(빨강) | 측정 방식 |
|---|---|---|---|
| CPU | 70% | 90% | 전체 CPU 사용률 |
| 메모리 | 80% | 90% | 사용 중 RAM / 전체 RAM |
| GPU | 80% | 90% | GPU 엔진 사용률 (Win10+ 내장 카운터, 벤더 무관) |
| 디스크 | 70% | 90% | 디스크 활성 시간 %(Disk Active Time) |
| 네트워크 | 70% | 90% | 현재 처리량 / 링크 속도 (NIC 대역폭 대비 %) |

디스크·네트워크는 절대 MB/s 대신 "링크 대비 사용률 %"로 정규화하여 색상 임계값을 일관되게 적용한다.

### D. 설정 (팝업 내 또는 트레이 메뉴)
- Windows 시작 시 자동 실행 on/off
- 위젯 위치·투명도 저장 (마지막 상태 복원)
- 임계값 커스터마이즈
- 시스템 트레이 아이콘 (좌클릭: 위젯 표시/숨김, 우클릭: 종료 등)

## 3. 기술적 접근

- **스택:** C# / .NET 8 (WPF), self-contained 배포로 런타임 설치 불필요한 단일 exe 생성
- **지표 수집:**
  - CPU/메모리/디스크/네트워크: `System.Diagnostics.PerformanceCounter` (Windows 내장 성능 카운터)
  - GPU: Windows 10+ "GPU Engine" 성능 카운터 카테고리 (`\GPU Engine(*)\Utilization Percentage`) —
    NVIDIA/AMD/Intel 벤더 구분 없이 동작, 별도 SDK(NVML 등) 불필요
- **업데이트 주기:** 1초 간격 (내부적으로 조정 가능하게 설계하되 기본값은 1초)
- **항상-위:** `Topmost=true` + 작업 표시줄에 아이콘이 뜨지 않는 도구창 스타일(`ShowInTaskbar=false`)
- **설정 저장:** 로컬 JSON 파일 (`%AppData%\SysMonWidget\settings.json`)에 위치/투명도/임계값/자동실행 여부 저장
- **자동 실행:** 레지스트리 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` 등록 방식 (관리자 권한 불필요)

## 4. 비기능 요구사항

- 위젯 자체 리소스 사용량: CPU 1% 미만, 메모리 수십 MB 이내 목표
- 관리자 권한 없이 실행 가능해야 함

## 5. 범위 제외 (이번 버전에서는 하지 않음)

- 프로세스별 리소스 랭킹(작업관리자 대체 기능)
- 원격 모니터링/멀티 PC 지원
- 클릭 통과(click-through) 모드
- 다국어 지원 (일단 한국어 또는 영어 단일)

## 6. 성공 기준

- 화면 구석에 위젯을 띄워두고, 부하 상황(예: CPU 90% 이상)이 발생했을 때 작업관리자를 열지 않고도
  3초 이내에 인지 가능해야 한다.
- 재부팅 후에도 위젯 위치/설정이 유지되어야 한다.
